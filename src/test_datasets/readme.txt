注意事项：
1. eng_param.csv每一行代表一个发射天线单元，当cell_id不同时视为完全独立的天线分别计算覆盖，当cell_id相同时取同cell下所有天线的仿真结果，在线性域进行相加
2. rs_power项可能包含误差，实际计算过程中建议先手动拟合。
3. azimuth为0时指向正北，90时指向正东；downtilt为0时指向水平方向，90时指向正下方地面。
4. power ratio为1时天线按照rs power功率发射，0.5时功率为一半，即rs_power - 3dB。
5. test_indoor.csv和test_outdoor.csv代表手机上报信号时处于室内或室外，合并时与test.csv等效，可优先评估纯室外的数据。
6. test内包含上报数据的cell_id，实际测试时需与基站cell_id对应，即每个cell_id单独评估误差。
